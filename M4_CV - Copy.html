LeafletJS

Pengertian

LeafletJS merupakan library berbasis JavaScript yang mampu dimanfaatkan untuk menampilkan peta interaktif pada halaman website dengan dibantu oleh beberapa fungsi yang sudah disediakan. Berikut berupakan komponen pada LeafletJS:

  • Map
  • Tilelayer
  • Marke
  • Popup
  • Event
  • Control
  • Vector/Feature Layer
  • Plugin
  • Service
